準備開業！

  • Post author:小編
  • Post published:October 13, 2020
  • Post category:Exam100Fun

其實開這網站是另有私心^^，係想俾個機會讀緊高中嘅阿仔 set up 和 maintain 一個商業網店，由租 server ，用 linix，起個 WordPress，申請 domain，都想佢有機會掂下，我又家窮四壁冇嘢可以俾佢賣，唯有賣我以前啲（有少少 proud of 嘅）自家製小學題目，不過整整吓覺得揾緊自己笨，一來組織番啲題目都有啲辛苦，成本效益超低，幾蚊雞一份卷賣到天光先回得返租 server 和申請 domain name 啲錢，二來條死仔都唔知有冇心機攪，我而家寫緊 blog 佢就係坐係我對面係度打緊 lol ，唉，希望佢大個之後知道爸爸媽媽幾用心良苦啦。。。
